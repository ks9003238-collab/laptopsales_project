# 💻 Laptop Price Prediction

An end-to-end Machine Learning project that predicts laptop prices based on various features such as company, CPU, RAM, operating system, and weight. The project includes data preprocessing, feature engineering, model training, and a Tkinter-based GUI for real-time predictions.

---

## 🚀 Features

* 📊 Data Cleaning & Preprocessing
* 🔧 Feature Engineering (PPI, Storage, Resolution, etc.)
* 🤖 Machine Learning Model (Random Forest Regressor)
* 📈 Model Evaluation (R² Score, MAE, RMSE)
* 🖥️ Tkinter GUI for user-friendly interaction
* ⚡ Real-time laptop price prediction

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Matplotlib
* Tkinter (GUI)
* Pickle (Model Saving & Loading)

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/laptop-price-predictor.git
```

2. Install dependencies:

```
pip install pandas numpy scikit-learn matplotlib pillow
```

3. Run the GUI:

```
python gui.py
```
